# Azure Traffic Manager (microsoft-azure-traffic-manager)
Azure Traffic Manager REST API enables DNS-based traffic load balancing for distributing traffic across global Azure regions. It supports configuring routing methods including priority, weighted, performance, geographic, and multivalue for high availability.
